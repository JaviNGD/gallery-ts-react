## Run Locally 

### Clone the project
```plaintext
git clone https://github.com/JaviNGD/gallery-ts-react.git
```

### Go to the project directory
```plaintext
cd gallery-ts-react
```

### Install dependencies
```plaintext
  npm install
```

### Start the server
```plaintext
  npm run dev
```

### See page in
```plaintext
  http://localhost:3000/
```
